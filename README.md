# DevCamper API

> Backend API for DevCamper application, which is a bootcamp directory website

## Usage

Rename ".env.example" to ".env" and update settings

```
cp .env.example .env
```

## Install dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in production mode
npm start
```
